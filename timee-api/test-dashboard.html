🌐 外部访问配置

要让前端应用从外部访问API，需要在Sealos中配置端口映射：

  1. 访问 Sealos控制台
  2. 进入DevBox管理，找到 'devbox-timee' 实例
  3. 配置端口映射：内部8080 → 外部8080
  4. 保存配置并重启实例

配置成功后，外部访问地址为: http://wmxkwzbmhlj.sealoshzh.site/api
